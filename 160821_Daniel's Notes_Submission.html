Daniel's Notes for the 1st Lesson


The first lesson has two main key contents

( 1) Development of technological empathy for what happens when a webpage is opened and
(2) Creation of a simple HTML document and the use of html tags

In Part I several key components behind a webpage are introduced, namely HTML, HTTP as well as The Internet and the basic function of servers. The hyperlinks provide overviews for the individual elements.

In Part II of the first lesson the following tags were interoduced:

<b> for making text bold
<em> for making text italic
<href> for making links
<img> for adding images and
<br> and <p> for creating whitespace

Later the difference between inline and block tags was explained, using the example of <span> vs <div>. Only block tags require closing tags.
Towards the end of the lesson there were some remarks on the overall document structure.
